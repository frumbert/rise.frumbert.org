This extension is a bit like a jigsaw puzzle - it allows you to unjumble any block-level object with an added time pressure.

It's not a SCORM interaction yet - still tinkering ðŸ˜Š

You can specify the number of divisions (4x3, 5x4, etc) and the time that the puzzle should be solved in (e.g. 2:35). A progress bar underneath the puzzle starts as soon as you start dragging. This changes colour as time passes to heighten the excitement.

The space between cells shrinks away when you solve the puzzle, and if you run out of time before solving it, you can reset and have another turn.
